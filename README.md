# MedVet - One-Health Platform

Welcome to **MedVet**, an integrated telemedicine platform designed to bridge the gap between human and animal healthcare. Our platform empowers families to manage their own medical needs and the health of their beloved pets seamlessly in one unified interface.

## 🚀 Live Demo

**Experience the platform here:** [MedVet Platform Demo](https://medvet-platform.vercel.app/)

---

## ✨ Key Features

- **Unified Patient Dashboard**: A single place to track your own health records, upcoming appointments, and your pet's profiles.
- **Find Specialists Easily**: Browse, filter, and book appointments with medical doctors and veterinarians in your area.
- **Smart Queue Tracking**: Live waiting room tracking so you always know your position during consultations.
- **Digital Prescriptions**: Securely access and download prescriptions for both humans and animals.
- **Integrated Pharmacy**: Order prescribed medicines and healthcare products directly through the platform.
- **Role-Based Portals**: Dedicated, feature-rich dashboards crafted specifically for Patients, Doctors/Vets, and System Administrators.

## 🛠 Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Custom modern CSS tailored for a highly responsive, intuitive, and accessible user experience.

## 📦 Getting Started Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm (or yarn/pnpm) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/medvet-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd medvet-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:3000` to see the application in action.
